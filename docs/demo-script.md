# 60 秒演示视频脚本

## 0-8 秒：首页

这是一个智能旅游规划 Agent。它不是聊天窗口，而是一个行程产品：用户填写旅行需求，系统直接生成可执行的每日路线。

## 8-22 秒：规划页

我在规划页输入出发地、目的地、日期、预算、节奏和偏好。提交后，后端会调用阿里云百炼的 OpenAI 兼容 Chat Completions，并要求模型返回结构化 JSON。

## 22-35 秒：生成结果

生成结果会按天展示，每一天都有时间段、地点、活动类型、费用估算和注意事项。成功结果会保存进 Supabase 的 4 张表。

## 35-45 秒：详情和重生成

打开详情页后，可以复制 Markdown，也可以按原条件重生成一份新计划。PDF 导出先作为后续功能占位。

## 45-53 秒：历史记录

历史页会从 Supabase 读取已保存的旅行计划。刷新页面或重启本地服务后，记录仍然存在。

## 53-60 秒：后台统计

后台页展示任务总数、成功率、平均耗时、失败日志和热门目的地，用来证明这个 Agent 已经形成从生成到保存再到运营观察的完整闭环。
